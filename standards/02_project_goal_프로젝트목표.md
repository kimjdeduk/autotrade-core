# 프로젝트 목표

이 프로젝트의 최종 목표는 자동매매 시스템 구축이다.

하지만 현재 단계의 우선 목표는 자동매매 실행 자체가 아니다.
현재는 자동매매를 안정적으로 만들 수 있는 기반 구조를 먼저 확립하는 단계다.

## 현재 우선 목표
1. OpenClaw 내부 LLM 역할 구조 확립
2. 시스템 구조 확립
3. 파일 구조 확립
4. 기존 스킬 및 기존 자산 재활용 기준 확립
5. 외부 개발 구조와 내부 운영 구조 분리

## 현재 단계에서 하지 말아야 할 것
- 처음부터 실주문 중심으로 설계하기
- 모든 기능을 한 번에 만들기
- 시스템 구축과 자동매매 전략 개발을 한 채팅에서 동시에 크게 진행하기
- OpenClaw 안에 본체 개발 로직을 전부 넣기

현재 프로젝트는 자동매매 프로젝트라기보다
자동매매 시스템 구축 프로젝트로 본다.
